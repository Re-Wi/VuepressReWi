# ReWi 怿窗の博客

## 把项目从gitee上克隆下来

git clone git@github.com:RejoiceWindow/vuepress-rewi.git

## 进入项目目录

cd vuepress-rewi

## 安装依赖

```shell
yarn  
# or
npm install
# or
cnpm install
```

## 启动项目

```shell
yarn dev 
# or
npm run dev
# or
cnpm run dev
```

## 其他

## 生成静态文件

```shell
yarn build 
# or
npm run build
# or
cnpm run build
```

## 运行发布命令:右键，Git Bush Here

```shell
yarn deploy 
# or
npm run deploy
# or
cnpm run deploy
```
